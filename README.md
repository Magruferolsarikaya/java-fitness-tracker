# Java Fitness Tracker (CLI)

A robust, console-based fitness tracking application built with Java. This project demonstrates core software engineering principles, including Object-Oriented Programming (OOP), Data Persistence (File I/O), and strict Exception Handling.

## 🚀 Features
* **Interactive CLI:** A user-friendly console menu to manage workouts, add exercises, and update user profiles dynamically.
* **Object-Oriented Design:** Strictly utilizes Encapsulation, Inheritance, and Polymorphism (Abstract `Exercise` class extended by `WeightExercise` and `CardioExercise`).
* **Data Persistence:** Automatically saves and loads user history to a local `.txt` file using Java's `BufferedReader` and `BufferedWriter`.
* **Bulletproof Input Handling:** Custom exception handling loops to prevent application crashes from invalid or unexpected user inputs.

## 🛠️ Technologies Used
* **Language:** Java (JDK 8+)
* **Core Libraries:** * `java.util.Scanner` (Input handling)
  * `java.io.*` (File operations)
  * `java.time.LocalDate` (Date tracking)
